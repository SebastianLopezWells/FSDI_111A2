SQL statements associated with CRUD operations.

SQL statement for create a new resource

The first letter of CRUD, 'C', refers to CREATE. In this operation, we use SQL INSERT INTO statement to insert a new record.t

Examples:

INSERT INTO "table_name" (column1, column2, ...)

VALUES (value1, value2, ...)

INSERT INTO persons (f_name, l_name, ...)

VALUES ("Eduardo", "Reynoso", ...)

We use this statement to add a new resource to a table named persons, we need to indicate what attributes we going to add with each values.

SQL statement for read a resource

The second letter of CRUD, 'R', refers to READ. In this operation, we use SQL SELECT statement to retrieve data or record-set from a listed table.

Examples:

SELECT * FROM 'TableName'

SELECT * FROM persons

SELECT * FROM persons WHERE f_name = "Eduardo"

We use this statement to read an existing resource from a table named 'persons', we need to indicate what are we going to read from that resource, in this case we use '*' to say that we are going to read every single attribute from every resource, we can specify what resource we want to desplegate using 'WHERE' and the attribute that we are going to use to filter the resource.

SQL statement for update a resource

The third letter of CRUD, 'U', refers to UPDATE. In this operation, we use SQL UPDATE statement to bring a change to an existing record(s) of the table.

Examples:

UPDATE 'TableName'

SET Column1=Value1, Column2=Value2,...

WHERE 'Expression'

UPDATE persons

SET f_name="Rodrigo", l_name="Perez",...

WHERE name=?

We use this statement to update an existing resource from a table named 'persons', we neeed to indicate what attributes are we going to change with each new value, and we need to indicate what resource is with 'WHERE'.

SQL statement for delete a resource

The last letter of CRUD, 'D', refers to DELETE. In this operation, we use SQL DELETE statement to delete the record(s) from the table.

Examples:

DELETE FROM 'TableName'

WHERE 'Expression'

DELETE FROM persons

WHERE name=?

We use this statement to delte an existing resource from a table named 'persons', we need to indicate what resource are we going to delete with 'WHERE'.
